About Context Variables

Context variables provide secure, encrypted storage for configuration values that your Hot code can access at runtime using ::hot::ctx/get.

All values are:

  • Automatically encrypted using AES-256-GCM
  • Validated as valid Hot code expressions
  • Scoped to environment or project level
  • Available in your event handlers and functions

Example usage in Hot code: 

api-key ::hot::ctx/get("API_KEY")
config ::hot::ctx/get("CONFIG")

Security: Context variable values are encrypted at rest and only decrypted when loaded by the worker for code execution.

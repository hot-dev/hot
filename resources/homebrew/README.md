# Homebrew Tap for Hot Dev

This is the official Homebrew tap for [Hot Dev](https://hot.dev),
a programming language and runtime for backend workflows.

## Installation

```bash
# Add the tap
brew tap hot-dev/hot

# Install Hot
brew install hot
```

Or install directly without adding the tap:

```bash
brew install hot-dev/hot/hot
```

## Updating

```bash
brew update
brew upgrade hot
```

## Uninstalling

```bash
brew uninstall hot
brew untap hot-dev/hot  # Optional: remove the tap
```

# Hot Dev Brand Guidelines

Hot Dev is open source and self-hostable, but the project name, logo, icon, and other brand assets identify the official Hot Dev project and services.

You may use the Hot Dev name and artwork to:

- Refer accurately to the Hot Dev project.
- Run, document, and distribute unmodified or reasonably modified self-hosted Hot Dev deployments.
- Link to the official project and explain compatibility.

Please do not use the Hot Dev name, logo, icon, or confusingly similar branding to:

- Present an unofficial service as the official Hot Dev service.
- Market a hosted or managed service in a way that suggests endorsement by Hot Dev.
- Remove required notices from the app in order to offer a white-label managed Hot Dev service.

Self-hosted deployments should keep the Hot Dev logo, sidebar project link, and copyright notice visible unless a separate written agreement says otherwise.

This document is a practical usage notice, not a trademark registration. The source code license is in `LICENSE`; brand assets and naming rights are separate from the Apache-licensed code.

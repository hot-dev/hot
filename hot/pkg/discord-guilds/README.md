# discord-guilds

discord guilds API bindings for Hot.

## Includes

This package covers: guild_role, guilds, prune, ban, guild

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/discord-guilds": "0.10.0"
```

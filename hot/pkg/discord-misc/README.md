# discord-misc

discord misc API bindings for Hot.

## Includes

This package covers: public, bot, lobby, answer, update, partner_sdk, poll, join, private, deprecated_create, leave, sticker, auto, bot_partner, delete, deprecated_delete, set, original, follow, linked, or, invite, consume, deprecated_list, active, all

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/discord-misc": "0.10.0"
```

# stripe-connect

stripe connect API bindings for Hot.

## Includes

This package covers: topups, external_accounts, payouts, linked_accounts, application_fees, transfers, accounts, account_sessions, link_account_sessions, account_links

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/stripe-connect": "0.10.0"
```

# stripe-misc

stripe misc API bindings for Hot.

## Includes

This package covers: account, fabric, country_specs, external_accounts, setup_intents, forwarding, linked_accounts, mandates, accounts, apple_pay, account_sessions, test_helpers, ephemeral_keys, setup_attempts, apps, link_account_sessions, account_links

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/stripe-misc": "0.10.0"
```

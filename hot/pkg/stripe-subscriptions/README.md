# stripe-subscriptions

stripe subscriptions API bindings for Hot.

## Includes

This package covers: coupons, subscription_schedules, prices, promotion_codes, subscription_items, plans, subscriptions

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/stripe-subscriptions": "0.10.0"
```

# stripe-reporting

stripe reporting API bindings for Hot.

## Includes

This package covers: balance_transactions, reporting, balance, exchange_rates, balance_settings, sigma

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/stripe-reporting": "0.10.0"
```

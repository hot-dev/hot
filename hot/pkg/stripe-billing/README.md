# stripe-billing

stripe billing API bindings for Hot.

## Includes

This package covers: billing_portal, invoice_payments, invoices, invoice_rendering_templates, invoiceitems, credit_notes, billing, quotes

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/stripe-billing": "0.10.0"
```

# stripe-payments

stripe payments API bindings for Hot.

## Includes

This package covers: sources, payment_methods, payment_method_domains, payment_intents, confirmation_tokens, payment_links, disputes, refunds, payment_attempt_records, payment_method_configurations, charges, payment_records

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/stripe-payments": "0.10.0"
```

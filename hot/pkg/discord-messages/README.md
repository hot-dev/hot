# discord-messages

discord messages API bindings for Hot.

## Includes

This package covers: thread, messages, crosspost, pins, message, pin, channel, typing

## Installation

Add this to the `deps` in your `hot.hot` file:

```hot
"hot.dev/discord-messages": "0.10.0"
```
